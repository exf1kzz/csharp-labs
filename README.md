# OOP Labs

Учебный проект с лабораторными работами по объектно-ориентированному программированию на C#.

В рамках проекта я отрабатывал базовые и продвинутые принципы ООП, проектирование доменной модели, разделение ответственности между слоями приложения и подходы к построению поддерживаемой архитектуры.
